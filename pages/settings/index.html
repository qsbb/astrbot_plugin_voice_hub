TTS 触发方式
TTS 后端

选择「AstrBot 内置 TTS」后，直接使用 AstrBot 已配置的 TTS 提供商合成语音，无需上传音频；MiMo 的音色克隆、情绪路由、AI 导演等功能不生效。

只有在「MiMo TTS」后端下才需要上传参考音频。切换到「AstrBot 内置 TTS」后可忽略音色库。

触发方式是唯一开关：概率触发会执行自动 TTS，LLM 自主决定只开放语音工具。

开启后插件会启动独立 HTTP 服务，可被任意 OpenAI TTS 客户端调用。token 和 model 均不校验，voice 字段填插件音色名/ID。

在 AstrBot 的 openai_tts 提供商配置中，Base URL 填入上方地址，API Key 填任意字符串（如 sk-anything）即可，否则会报 Missing credentials。

诊断会生成一段短音频，用来检查 Key、模型、音色和网络。
